### 2. **BreadcrumbList Schema**
```json
{
  "@type": "BreadcrumbList",
  "itemListElement": [
    { "position": 1, "name": "Home" },
    { "position": 2, "name": "Play Games" }
  ]
}
```
**Purpose**: Shows navigation path in search results

### 3. **CollectionPage Schema**
```json
{
  "@type": "CollectionPage",
  "name": "Free Online Games Collection",
  "hasPart": [
    { "@type": "Game", "genre": "Puzzle" },
    { "@type": "Game", "genre": "Action" },
    // ... 5 game types
  ]
}
```
**Purpose**: Indicates this is a collection of games

### 4. **Offer Schema**
```json
{
  "@type": "Offer",
  "price": "0",
  "priceCurrency": "USD",
  "availability": "InStock"
}
```
**Purpose**: Shows games are free in search results

### 5. **FAQPage Schema**
```json
{
  "@type": "FAQPage",
  "mainEntity": [
    { "Question": "Are these games really free?" },
    { "Question": "Do I need to download anything?" },
    // ... 5 questions total
  ]
}
```
**Purpose**: Displays FAQ rich snippets in search results

```json
{
```
